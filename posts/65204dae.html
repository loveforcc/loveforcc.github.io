1.GO语言简介

Go 语言，我们可以简单的写成 Golang.

1.1GO发展

2007 年，谷歌工程师 Rob Pike, Ken Thompson 和 Robert Griesemer 开始设计一门全新的语言，这是

Go 语言的最初原型。

2009 年 11 月 10 日，Google 将 Go 语言以开放源代码的方式向全球发布。

2015 年 8 月 19 日，Go 1.5 版发布，本次更新中移除了”最后残余的 C 代码”

2017 年 2 月 17 日，Go 语言 Go 1.8 版发布。

2017 年 8 月 24 日，Go 语言 Go 1.9 版发布。 1.9.2 版本

2018 年 2 月 16 日，Go 语言 Go 1.10 版发布。

1.2GO语言特点

  • 简介

    • Go 语言保证了既能到达静态编译语言的安全和性能，又达到了动态语言开发维护的高效率使用一个表达式来形容 Go 语言：Go = C + Python , 说明 Go 语言既有 C 静态语言程序的运行速度，又能达到 Python 动态语言的快速开发。

  • 引入包的概念 ：用于组织程序结构，Go语言的一个文件都要归属于一个包，而不能单独存在。

  • 垃圾回收机制，内存自动回收，不需开发人员管理 【稍微不注意就会出现内存泄漏】

  • 天然并发【重要特点】goroutine

    • 从语言层面支持并发，实现简单
    • goroutine，轻量级线程，可实现大并发处理，高效利用多核。（PS协程）
    • 基于CPS并发模型（Communicating Sequential Processes）实现

  • 吸收了管道通信机制，形成go语言特有的管道channel，通过管道channel，可以实现不同的goroute之间的相互通信

  • 函数返回多个值

  • 新的创新：比如切片slice，延时执行defer等

1.3GO环境配置

1.3.1 前往GO中文网下载SDK

GO中文网 ：https://studygolang.com/ 下载window版镜像

1.3.2配置环境

GO环境中三个环境需要配置

  • PATH ：path为win环境配置该环境作用让GO命令在任何dos路径下都可以运行（相当于快捷方式）
  • GOROOT：GO SDK安装根路径
  • GOPATH：工作路径

环境配置完成后可以使用 go -v 测试配置是否成功

注意在GO 1.14版本后使用了GO Module模式管理

3.2变量

变量相当于内存中一个数据存储空间的表示，你可以把变量看做是一个房间的门

变量三要素 ：变量=变量名+值+数据类型

在GO中变量分为全局变量和局部变量

  • 全局变量 声明在方法体外变量

  • 局部变量 在方法体中声明变量

需要注意以下几点：

  • Golang 的变量如果没有赋初值，编译器会使用默认值, 比如 int 默认值 0 string 默认值为空串， 小数默认为 0
  • 变量在同一个作用域(在一个函数或者在代码块)内不能重名
  • 该区域的数据值可以在同一类型范围内不断变化(重点)

3.6运算符

运算符是一个在任何一门语言中都是常见的知识：算术运算符、关系运算符、逻辑运算符、位运算符、赋值运算符

本知识点只讲GO与其他语言不同的部分

3.6.1 自增自减

在编程语言中 自增自减 是非常讨厌的语法 i++、++i、–i、i-- 这些使用起来都是令阅读人员讨厌的。

在GO中 ++、-- 并不是运算符，而是单独的语法 并且只能出现在变量的后面

GO只支持 i++、i-- ，Java等语言支持 i++、++i、–i、i–。

3.6.2 + 加号 、字符串连接符

在GO中 运算符两边都是 相同数值型是 为 运算符 ，在两边都是字符串时为拼接符（连接符）(PS GO中运算时没有自动类型转换，后面单独一节讲解 GO中类型转换和计算)

在Java中 +号两边是数值型 为 运算符 ，只要有一边为字符串为字符串拼接符
